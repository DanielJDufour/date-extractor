# date-extractor
date-extractor helps you extract dates from text

# Installation
```
pip install pytz
pip install date-extractor
```

# Use
```
from date_extractor import extract_dates
text = "I arrived in that city on January 4, 1937"
dates = extract_dates(text)
```
